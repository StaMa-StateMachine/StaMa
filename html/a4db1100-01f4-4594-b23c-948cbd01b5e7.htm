Shape Types

Introduces the different StaMa Visio Shapes and explains their functionality.

StaMa Visio Shapes are set of Microsoft Visio shapes that serve to assemble an arbitrary deep nested hierarchical state machine and generate the structural code for a StaMa based implementation of the state machine.

The shapes can be found in the Document Stencil window when creating a drawing from the StaMaShapes.vst template. The general concept is to drag the shapes from the Document Stencil into the state diagram and position and connect them there.
